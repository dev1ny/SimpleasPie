# 商超利润计算器 Logo 改版设计

日期：2026-04-11

## 背景

当前应用图标过于简单，缺少“计算器”和“利润分析”这两个核心识别点。新图标需要在 macOS 和 Windows 的桌面、安装器、任务栏和 Finder/资源管理器里都更清楚、更专业，同时保留经典工具软件的稳定感。

本次改版只调整应用 logo 与其派生图标资源，不改动应用界面布局、文案和计算逻辑。

## 目标

- 让图标一眼看出“计算器”属性
- 在不复杂化图标的前提下，体现“利润增长/利润分析”含义
- 保持扁平化、经典、商务化风格
- 保证小尺寸下仍有足够识别度
- 替换现有打包资源并重新发布桌面安装包

## 已确认方向

### 视觉方向

- 风格：扁平化、经典桌面工具图标
- 概念：`计算器 + 利润箭头`
- 布局：`显示屏箭头`
- 配色：`墨蓝 + 浅灰 + 亮橙`

### 设计取舍

- 不做立体高光、玻璃拟物或复杂阴影，避免图标显旧或缩小后发糊
- 不在图标中加入文字、`¥`、百分号等额外符号，避免信息过载
- 不使用跨越整个图标的大箭头，避免破坏“经典计算器”轮廓

## 最终图标方案

### 构图

- 底板为圆角方形图标画布
- 中心主体为一台经典计算器
- 计算器顶部为浅灰色显示屏
- 显示屏中放置一根简洁的上升箭头，表示利润增长
- 下方为规整的按键网格
- 右下角使用亮橙色等号键，强化“计算器”识别

### 颜色

- 机身主色：墨蓝，用于主体轮廓和大面积底色
- 显示屏：浅灰或偏冷白，提高与箭头的对比
- 利润箭头：比显示屏更深的蓝或偏青蓝，确保在小尺寸下仍清楚
- 强调键：亮橙，用于等号键，增加记忆点

### 形态要求

- 所有元素使用简单几何图形构成
- 线条和圆角适中，避免过细
- 显示屏箭头使用单一上升方向，不做多段复杂折线
- 按键数量以清晰为优先，不强求真实键位

## 小尺寸适配要求

- 以 `16x16`、`32x32` 和任务栏尺寸为主要约束
- 箭头必须在缩小后仍能辨认出方向感
- 按键区允许轻微简化，但必须保留“计算器按键区”的整体印象
- 不依赖细描边、细阴影或低对比色块来表达关键结构

## 实施范围

### 需要修改的文件

- `src-tauri/icons/128x128@2x.png`

以下文件由脚本重新生成：

- `src-tauri/icons/32x32.png`
- `src-tauri/icons/128x128.png`
- `src-tauri/icons/icon.ico`
- `src-tauri/icons/icon.icns`

### 使用现有链路

- 保持 `scripts/generate-icons.py` 作为图标派生资源生成入口
- 以新的 `128x128@2x.png` 为源文件生成其余平台图标

## 发布范围

完成图标替换后，需要重新执行：

- macOS 通用包构建
- Windows x64 安装包构建
- Git 提交与推送
- 更新 GitHub Release，使下载页带上新图标版本的安装包

## 成功标准

- 应用图标在本地文件系统中能明显看出“计算器 + 增长”含义
- `32x32` 和更小尺寸下仍保持可识别
- `.ico` 和 `.icns` 正常生成
- macOS 和 Windows 打包脚本都能跑通
- GitHub Release 更新成功，下载包中携带新图标

## 非目标

- 不改应用名称
- 不改欢迎页、窗口标题或界面配色
- 不新增品牌字标或完整 VI 系统
